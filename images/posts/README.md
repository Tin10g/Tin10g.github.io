# Post Images

这个目录专门用来存放文章里的图片。

推荐按文章再分子目录，例如：

- `static/images/posts/git-upload/demo1.png`
- `static/images/posts/about/profile.jpg`

在 Markdown 里这样引用：

```md
![示例图片](/images/posts/git-upload/demo1.png)
```

如果是 `content/posts/git上传项目.md`，推荐把它的配图放到：

- `static/images/posts/git-upload/`

这样后续整理会更清晰。
